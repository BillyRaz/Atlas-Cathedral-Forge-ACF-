# ACF Version Details

## Product

Atlas Cathedral Forge (ACF)

## Release Name

ACF Core Release

## Release Stage

Core Complete

## Version Summary

This release establishes the core ACF editor workflow for:

- scene scanning and categorization
- blockout prefab creation
- floor snapping and layout planning
- wall and roof generation
- corridor-aware seam handling
- mask-floor support for multi-floor layouts
- live multi-object editing
- final prefab replacement
- deep diagnosis and generation reporting

## Included Systems

- category utility system
- object data metadata system
- toolbar and editor helpers
- key and door data linking
- generated object visibility controls
- roof-to-mask-floor conversion workflow

## Primary User Outcome

Users can now build, organize, edit, diagnose, and prepare complete blockout-based level layouts inside Unity using a single integrated ACF workflow.
